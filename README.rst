####################
Render Titans Videos
####################

This package provides a command-line interface (cli) to batch-render Titans Of
Eden videos that were created in blender.

**********
Quickstart
**********

#. Requirements:

   #. python3.5+
   #. ffmpeg

#. Get blender:

   .. code-block:: bash

      ./install-blender

#. Render all:

   .. code-block:: bash

      ./render
